# Requirements
Symfony CLI -> https://symfony.com/download
Composer -> https://getcomposer.org/
Node v20.16.0

# API Symfony 7.2
```
cd api/
composer install
symfony server:start
curl -X GET http://localhost:8000
```

# FRONT Angular v19
```
cd front/
npm install
npm start
curl -X GET http://localhost:4200
```
